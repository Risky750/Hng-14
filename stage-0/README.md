# Todo Card Component

A clean, modern, and accessible Todo / Task Card built with HTML, CSS, and Vanilla JavaScript.
This project focuses on semantic HTML, accessibility, responsiveness, and testability.

## Features
- Task title, description, and status display
- Live time remaining countdown (updates every 60 seconds)
- Priority badges (Low / Medium / High)
- Tag system for categorizing tasks
- Checkbox to mark task as complete
- Edit and delete action buttons (dummy handlers included)
- Fully accessible (ARIA labels, semantic HTML)
- Fully responsive (mobile → desktop)
- Testing-friendly with required data-testid attributes

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
- index.html -> Main UI structure
- styles.css -> Styling
- script.js -> Behaviour & logic
- README.docx -> Documentation
